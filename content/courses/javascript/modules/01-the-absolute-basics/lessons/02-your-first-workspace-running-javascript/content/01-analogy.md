Think of the code editor as your workshop - it's where you'll build and test your code. Just like a carpenter has a workbench where they can quickly test if a joint fits or a piece is the right size, Code Tutor is where you can quickly try out JavaScript code and see the results immediately.

Code Tutor runs JavaScript using modern runtimes like Bun or Node.js - the same JavaScript engines that power web browsers, but running on your computer. This means you can write and test JavaScript code without needing a web browser. The `console.log()` function works exactly the same way!

**To run JavaScript, you'll need a JavaScript runtime installed. We recommend Bun:**

1. Download Bun from https://bun.sh
2. Install it following the platform-specific instructions
3. Alternatively, you can use Node.js from https://nodejs.org
4. Restart Code Tutor

Once installed, write your code and click 'Run Code' to see the results!
